## 一、本版本定位与目标

### 1.1 从 v0.4 回退的原因

v0.4（非归一化 softplus 头）在 GCR 盲测中 EMD 降至 5.30（四版最低），但**峰位误差从 v0.3 的 5 bin 涨至 8 bin（四版最差）**。根因为：

1. `composite_wide` 选择指标中峰位权重仅 0.1，优化器牺牲峰位换取整体 EMD 下降
2. 非归一化头 + log 压缩目标的逆变换放大了峰区位置偏移

### 1.2 v0.5 核心目标

| 优先级 | 目标 |
|--------|------|
| **第一** | 峰位对齐（peak position alignment） |
| **第二** | 0.1–1000 keV 主体谱线整体吻合 |
| 不追求 | 高能尾部（>1000 keV）快速归零 |

## 二、核心技术变更

### 2.1 回退归一化头

- 放弃 v0.4 的 `softplus`（非归一化），回到 `softplus_renorm`（归一化）
- 目标谱随之改回纯密度（不做 log 压缩），由 `build_preprocessors()` 自动匹配
- `test_preprocessor_head_match` 锁定配对关系，防止 v0.1 类型错配

### 2.2 新增：软峰位损失（`soft_peak_loss`）

```python
def soft_peak_loss(pred, target, window_half=30):
    peak_idx = target.argmax(dim=-1)          # 目标峰位（不参与梯度）
    # 在目标峰位 ±30 bin 窗口内，计算 pred 和 target 的软质心
    # 惩罚两者质心的平方偏差
```

- **设计动机**：用窗口局部质心而非全谱 soft-argmax，避免高能平底（~1e-4）把质心拉偏
- **权重**：`w_peak: 2.0`，与 `w_emd: 2.0` 同量级，峰位作为一等优化目标
- **可微性**：相对于预测 pred 完全可微，梯度直接约束峰位

### 2.3 新增：区间加权 MSE（`region_weight`）

```yaml
loss:
  region_kev: [0.1, 1000.0]
  region_in_weight: 1.0      # 主体谱线区（0.1–1000 keV）权重
  region_out_weight: 0.1     # 高能尾部（>1000 keV）权重压低至 10%
```

- 优化器聚焦 0.1–1000 keV 主体区间，高能平底区不强求改善
- `build_region_weight(energy_grid, region_kev)` 构造逐 bin 权重向量

### 2.4 新增：selection_score 选择指标

```
selection_score = peak_err + 0.3 × region_emd[0.1-1000 keV]
```

- 主判据：留出集峰位误差（bin 数）
- 次判据：0.1–1000 keV 区间内的 EMD（排除高能平底干扰）
- 彻底取代 v0.4 的 `composite_wide`（w_peak=0.1 过小）
- 取代 v0.2/v0.3 的 `val_emd`（与 GCR EMD 脱钩问题已记录）

### 2.5 数据扩展：16 能量（分两阶段）

| 阶段 | 训练能量集 | 训练次数 |
|------|-----------|---------|
| 阶段 1（v0.5 初） | 13 个（原有，含 90/100 MeV） | 1 次 |
| 阶段 2（+3 能量） | 16 个（新增 5000/6000/8000 MeV） | 1 次 |

留出能量沿用 v0.4：`[100, 600, 2000, 7000]`（不变）

### 4.2 谱线图观察

**阶段 2（16 能量）谱线**：
- 预测谱（红色虚线）呈现明显**双峰结构**：
  - 左侧主峰：~3–4 keV（与真实峰位置接近）

## 五、关键发现与问题分析

### 5.1 峰位损失的作用与局限

**有效之处**：
- 留出集峰位误差降至 3–4 bin（训练过程稳定），明显优于 v0.4 的 8 bin
- selection_score 在留出集上的代理信号比 composite_wide 更准确

**局限**：
- 阶段 1（13 能量）在盲测中峰位误差仍为 8 bin，与 v0.4 相同
- **留出集峰位 ≠ 部署谱峰位**：soft_peak_loss 约束的是"留出 4 个能量的单能谱峰位"，而部署谱是宽谱混合，峰位受不同能量权重分布影响，与单能留出集的峰位特性不同

### 5.2 新增 5000/6000/8000 MeV 的影响

**物理背景**（用户指出，更正我的分析）：
> 在 SiC 探测器中，随质子/中子能量升高，能量沉积峰位**向左（低 keV）移动**。

因此：
- **高能谱（5000–10000 MeV）**：TEPC 响应峰位在**低 keV**（~2–4 keV）
- **低能谱（90–500 MeV）**：TEPC 响应峰位在**高 keV**（~10–50 keV）

**双峰成因（修正后分析）**：
- 左侧主峰（~3 keV）：高能训练谱（2000–10000 MeV）的贡献
- 右侧次峰（~10–50 keV）：**低能训练谱（90–500 MeV）被过度估计**

新增的 5000/6000/8000 MeV 谱使得高能端更密集（TEPC 峰在 ~2–3 keV），反而把主峰拉得更靠左，与真实峰（~4–5 keV）的偏差可能增大，导致 EMD 恶化（5.30 → 6.32）。

### 5.3 核心瓶颈：低能分量过度估计

**现象**：部署谱的预测在 ~10–50 keV（低能谱 TEPC 响应区）出现次峰，而真实 TEPC 在此区间接近零。

**假设**（待验证）：模型对 90–500 MeV 低能训练谱在部署谱中的贡献过度估计。

**可能的根因**：
1. 合成训练族（`dirichlet_uniform`, `loguniform`）对所有 12 个训练能量进行宽幅混合，低能谱与高能谱出现频率相等，但部署谱中低能中子贡献可能很小
2. 没有约束低能分量权重的机制，模型自由分配权重给低能谱，导致右侧次峰

## 七、待解决的核心问题

### 问题 1：双峰结构（最高优先级）

| 属性 | 描述 |
|------|------|
| 现象 | 预测谱在 ~10–50 keV 出现次峰，真实谱中此区间近零 |
| 假设根因 | 低能训练谱（90–500 MeV）贡献被过度估计 |
| 验证方法 | 单独查看 90/100/200/300 MeV 的 TEPC 谱，确认其峰位是否在 10–50 keV |
| 潜在方向 | 减少低能谱在合成混合中的权重；或对低能谱的权重引入稀疏约束 |

### 问题 2：峰位误差代理信号不准

| 属性 | 描述 |
|------|------|
| 现象 | 留出集峰位误差 3–4 bin，但盲测 8 bin（阶段 1） |
| 根因 | 留出集仅含 4 个单能混合；部署谱是宽谱，峰位由能量权重分布决定，与留出集峰位不等价 |
| 潜在方向 | 构造更宽广的留出集验证谱；或直接建立能量权重分布与 TEPC 峰位的解析关系 |

### 问题 3：高能数据（5000/6000/8000 MeV）加入后 EMD 变差

| 属性 | 描述 |
|------|------|
| 现象 | 增加三个高能锚点后，EMD 从 5.30 升至 6.32 |
| 假设 | 高能谱 TEPC 峰在 ~2–3 keV（更左），与真实峰 ~4–5 keV 方向相反，模型被拉偏 |
| 验证方法 | 查看 5000/6000/8000 MeV 的 TEPC 峰位，确认是否比真实部署谱峰位更靠左 |

## 八、下一步建议（v0.6 方向）

> 注：以下方向均基于留出集分析和物理推断；**不得**以 v0.5 盲测结果直接调整超参数。

### 方向 A：解决低能分量过度估计

1. **验证**：绘制各单能 TEPC 谱，确认 90–500 MeV 峰位确实在 10–50 keV
2. **合成修改**：在合成混合时对低能谱（< 500 MeV）进行下采样或权重压制
3. **物理约束**：若部署谱能量成分偏高能，则合成族应减少低能谱参与比例

### 方向 B：评估 5000/6000/8000 MeV 数据是否有益

1. 比较 13 能量 vs 16 能量在留出集上的 peak_err 和 region_emd
2. 若新能量使留出集峰位变差（高能谱峰位偏左，拉偏整体预测），则从训练集移除
3. 若新能量有益于某些留出能量，保留但调整合成权重

### 方向 C：改进代理指标

- 扩展留出集为"多能量宽谱混合"而非单能采样
- 使 selection_score 的峰位误差更接近部署谱场景
